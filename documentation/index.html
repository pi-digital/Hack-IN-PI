Installation

Follow the steps below to get started with your Site Template:

  • Open the Package/Euthenia template Folder to find all the Templates Files
  • You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  • Make sure you upload the required files/folders listed below:
    • Euthenia template/css – Extra Stylesheets Folder
    • Euthenia template/img – Images Folder
    • Euthenia template/js – Javacripts Folder
    • Euthenia template/index.html – Index File/Homepage
  • You’re now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Logo Settings

The Logo Container can be found in div with '.logo-wrap' class


	<div class="logo-wrap">
		<img src="img/logo.png" alt="">
	</div>

Retina Logo and other images

If you wish to use a Retina Logo, or any other image, make sure that you have double the size of your Standard image in the "img" folder.

Changing Fonts

You can find the Linking to the Font Files in the head tag of all the .html files.

Template Customization

We have include a style CSS File in the css Folder so that you can better handle your Customizations while updating Euthenia to the Latest Version. Simply add all your Custom CSS Codes in the css/style.css File and link it in the Document <head>. Also make sure that this is the Last Linked CSS File in the Document <head> so that your Custom CSS Styles are Overwritten properly.
